RaceClocker Timing Usage

Parameter Value
Location Start or Finish. E.g. Location=Start
PrimaryID The ID of the results of the primary race.
https://raceclocker.com/b4e0c99a => b4e0c99a is the ID. E.g. PrimaryID=b4e0c99a
SecondaryID The ID of the results of the secondary race.
https://raceclocker.com/b4e0c99b => b4e0c99b is the ID. E.g. SecondaryID=b4e0c99b
Number The number of last timed results to be shown. E.g. Number=10
Refresh Refresh/reload time of the page in seconds. E.g. Refresh=5
MaxDeviation Maximum deviation between the primary and secondary time in seconds.
The result will show red if this is exceeded. E.g. MaxDeviation=0.5
Example URL https://yourwebsite.com/timing/index.php?Location=Start&PrimaryID=b4e0c99a&SecondaryID=b4e0c99b&Refresh=10&MaxDeviation=0.5
